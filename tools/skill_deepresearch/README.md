# 深度搜索技能

基于多轮搜索与信息整合的深度研究技能，支持方法追踪、来源评估与迭代深度分析，适用于需要全面调研的场景。

数据来源：[SkillHub 腾讯](https://skillhub.cloud.tencent.com/skills/in-depth-research-1-0-0)，作者：nickolaslab

## 一、功能说明

- **多源调查**：从多个来源收集信息，确保调研的全面性和客观性
- **方法追踪**：记录并追踪研究过程中使用的方法和路径
- **来源评估**：对信息来源的可信度和权威性进行评估
- **迭代深度**：支持多轮迭代搜索，逐步深入挖掘主题

## 二、适用场景

当用户需要以下类型的调研时使用此技能：

- 行业分析、市场调研、竞品分析
- 技术选型、方案对比、可行性研究
- 学术调研、政策解读、舆情分析
- 任何需要全面、深入、多角度信息收集的场景

## 三、使用方法

### 基础用法

直接向智能体描述你的研究需求，例如：

> 帮我深度调研一下目前国内大模型厂商的竞争格局

### 进阶用法

可指定研究维度和要求：

> 请从技术路线、商业模式、融资情况三个维度，深度调研国内 AI Agent 创业公司

## 四、技能文件说明

该技能包含以下核心文件：

| 文件 | 说明 |
| :--- | :--- |
| `SKILL.md` | 技能主文件，定义研究流程和方法论 |
| `methodology.md` | 研究方法论，规范搜索和分析策略 |
| `sources.md` | 来源管理，评估信息来源的可信度 |
| `output-formats.md` | 输出格式规范，确保报告结构化 |
| `_meta.json` | 技能元数据 |

## 五、输出示例

深度搜索技能的输出通常包含：

1. **研究概述**：研究目标、范围和方法
2. **核心发现**：分主题整理的关键信息
3. **来源评估**：各信息来源的可信度评级
4. **对比分析**：多维度对比和分析
5. **结论与建议**：基于调研结果的总结和建议
6. **参考来源**：完整的引用来源列表
