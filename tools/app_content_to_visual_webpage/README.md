# 内容秒变可视化网页智能体

## 一、应用描述

本应用依托 AI 能力和 Korx MCP 工具，用户无需手动编写前端代码，仅需用自然语言描述要求，然后上传要生成的文档/表格，AI 智能体便会调用 MCP 工具生成可分享的可视化网页，降低系统操作门槛，适配销售、客服等岗位的业务需求。

## 二、应用功能

内容秒变可视化网页智能体应用具有以下功能：
- 生成可视化网页：根据用户上传的文档/表格，调用Korx MCP 工具将 AI 编写的 html 代码变成可分享的可视化网页。
