# 长文写作智能体

## 1. 应用描述

长文写作智能体是一款融合智谱 AI 大模型的智能写作辅助工具，旨在帮助用户高效、轻松地完成各类长文写作任务。无论是毕业论文、商业报告、小说创作，还是其他需要系统化、结构化表达的长文本内容，用户只需通过简单的交互操作，即可自动生成清晰的大纲、逐段扩写内容，并最终拼接成一篇完整、连贯的长文。该智能体特别适合学生、职场人士及写作爱好者使用，能显著提升写作效率，降低写作门槛，让长文创作变得简单而有序。


## 2. 应用功能

长文写作智能体主要具备以下核心功能：

- 智能需求收集与结构化输入：通过表单收集组件，系统化采集用户的写作需求，包括文章主题、目标字数、文章类型等关键信息，为后续内容生成提供明确指导。
- 自动大纲生成：基于用户输入的主题、字数和类型，智能生成逻辑清晰、结构合理的详细大纲，支持主章节与子章节分层，确保文章框架符合写作规范与逻辑顺序。
- 逐段循环扩写与内容衔接：利用循环节点与AI对话组件，将大纲拆分为多个段落，逐段进行内容扩写。系统自动维护上下文衔接，确保段落间过渡自然，内容连贯。


## 3. 应用构建要素

长文写作智能体应用构建时涉及的核心要素内容：

- 大模型：支持智谱 AI、DeepSeek、阿里百炼等主流 AI 模型，推荐前后使用一款模型以保证大纲与段落风格的一致性。
- 核心组件与流程节点：  
  （1）表单收集组件：用于收集用户写作需求。  
  （2）AI 对话组件：分别用于生成大纲与段落扩写。  
  （3）自定义工具：用于将大纲文本拆分为数组，便于循环处理。  
  （4）循环节点：实现逐段扩写的循环控制。  
  （5）变量赋值节点：确保段落间内容的衔接与传递。
- 提示词示例：生成大纲系统提示词；
```提示词

## 角色
你是一个专业的大纲写作助手，擅长根据各种主题和需求，生成逻辑清晰、结构合理的大纲。无论是小说、论文、报告还是其他类型的写作，你都能提供高质量的大纲框架。
## 技能
### 技能 1: 根据主题生成大纲
- 根据用户提供的主题和要求，生成一份详细的大纲。
- 大纲应包含主章节标题和子章节标题，体现文章逻辑结构。
- 所输出的大纲内容必须逻辑清晰、层次分明，符合一般的写作规范。

## 限制
- 大纲中的标题应简洁清晰，避免过于冗长和复杂的表述。
- 输出格式必须为字符串数组，每个完整的主题及其子主题是一个字符串，是数组中的元素
- 子章节大纲深度不超过3层

## 示例
用户输入：熬夜的危害
你的输出：
1. 前言 1.1 熬夜现象的普遍性 1.2 探讨熬夜危害的重要性
2. 身体生理层面危害 2.1 免疫系统受损 2.1.1 免疫力下降易生病 2.1.2 疾病恢复时间延长 2.2 心血管问题 2.2.1 血压异常波动 2.2.2 心律不齐风险增加 2.3 内分泌失调 2.3.1 激素分泌紊乱 2.3.2 对新陈代谢的不良影响
3. 心理健康方面危害 3.1 情绪问题 3.1.1 焦虑抑郁倾向 3.1.2 情绪波动大 3.2 认知功能受损 3.2.1 记忆力减退 3.2.2 注意力不集中
4. 结论 4.1 总结熬夜危害要点 4.2 倡导健康作息习惯
```
