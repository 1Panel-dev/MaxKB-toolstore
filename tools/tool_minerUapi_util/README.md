# 在线 MinerU API 解析 PDF 工具

## 工具简介
本工具集成 **MinerU** 官方 API，专门用于解决 RAG 场景中 PDF 解析质量差的痛点。它可以处理多栏布局、数学公式和表格，将 PDF 高精度转化为 **Markdown**，显著提升大模型的检索准确率。

---

## 核心功能
* **智能布局分析**：精准识别标题、段落、列表，自动剔除页眉页脚干扰。
* **公式与表格识别**：将复杂公式转为 LaTeX，表格转为标准 Markdown 格式。
* **OCR 强力加持**：支持扫描件、乱码 PDF 识别，兼容多国语言。
* **结构化输出**：直接输出适合 LLM 处理的 Markdown 纯文本。

---

## 参数配置

### 1. 启动参数（环境变量）
| 参数名 | 组件类型 | 必填 | 描述                                                                                 |
| :--- | :--- | :--- |:-----------------------------------------------------------------------------------|
| `api_token` | 密码框 | 是 | MinerU 平台的 API Token。可在 [MinerU 官方 API 管理](https://mineru.net/apiManage/token) 创建。 |
| `maxkb_base_url` | 文本框 | 是 | **关键参数**：MaxKB 的外网访问地址，用于 MinerU 服务器远程下载待解析文件。                                     |

### 2. 输入参数
| 参数名 | 数据类型 | 必填 | 描述 |
| :--- | :--- | :--- | :--- |
| `pdf_file_list` | array | 是 | 引用参数，MaxKB 自动传入的文件列表。 |

---

## 工作流程
1. **任务提交**：将 PDF 外网下载链接推送至 MinerU 接口。
2. **异步轮询**：每 3 秒检查一次云端解析进度。
3. **文本提取**：任务完成后自动下载结果包，并解压提取其中的 `.md` 文本内容。

---

## ⚠️ 使用限制与注意事项
为了确保您的工具正常运行，请务必关注以下信息：

* **单文件解析限制**：**目前版本一次仅支持解析列表中的第一个 PDF 文件**，如有多个文件请分次调用。
* **网络访问要求**：请确保您的 `maxkb_base_url` 地址在**外网可直接访问**。若 MaxKB 部署在内网且未做穿透，MinerU 官方服务器将因无法下载文件导致任务失败。
* **图片提取说明**：**本工具目前仅返回解析后的 Markdown 文本内容**，暂不支持提取并展示 PDF 中的原始图像。
* **API 接口声明**：当前调用的官方接口为 `https://mineru.net/api/v4/extract/task`。若官方接口协议变更，请关注工具版本升级。
* **耗时与额度**：解析耗时取决于 PDF 页数与复杂度。请确保您的 MinerU 账户余额充足。
