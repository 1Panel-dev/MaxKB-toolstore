# 中文公文生成智能体

## 1. 应用描述

中文公文生成智能体是一款专为党政机关、企事业单位打造的智能公文写作辅助工具。基于大语言模型，严格按照《党政机关公文处理工作条例》及 GB/T 9704-2012《党政机关公文格式》国家标准，自动生成规范、标准的中文公文。用户只需提供基本的公文要素（如发文机关、文种、事由等），系统即可快速生成格式规范、内容完整的公文文本，大幅提升公文起草效率，降低格式错误风险。

## 2. 应用功能

- **智能意图识别**：自动判断用户输入是"公文生成任务"还是不明确的需求，非公文任务会给出友好提示。
- **知识库检索增强**：支持对接知识库，检索与用户主题相关的参考资料，提升公文内容质量。
- **双模式内容生成**：
  - **完整内容型**：用户粘贴已有正文，系统保留原文主体，仅做必要的结构整理和标点规范。
  - **主题生成型**：用户只输入主题，系统自动生成完整公文正文。
- **多文种支持**：通知、通报、报告、请示、批复、函、纪要、决定、意见、议案、心得体会、总结、发言稿等。
- **公文元信息自动提取**：自动识别标题、文种、主送机关、落款、日期、红头机关、发文字号、份号、密级、紧急程度、签发人、抄送、印发机关、印发日期、附注、附件等字段。
- **国标排版 Word 导出**：调用公文排版技能生成符合 GB/T 9704-2012 标准的 Word 文件。
- **MaxKB OSS 文件上传**：生成的 Word 文件通过 MaxKB 平台的 OSS 接口上传，返回 30 分钟有效的临时下载链接。

## 3. 应用构建要素

中文公文生成智能体应用构建时涉及的核心要素内容：

- **大模型**：支持 DeepSeek、智谱 AI、阿里百炼等主流大语言模型。

- **工作流节点**：
  1. **意图识别节点**：分类用户输入为"公文生成任务"/"需求不明确"/"其他"。
  2. **知识库检索节点**：embedding 检索模式，top_n=3，相似度阈值 0.5。
  3. **AI 对话节点（公文内容准备）**：支持完整内容型和主题生成型两种处理模式，输出结构化的公文元信息和正文。
  4. **AI 对话节点（公文 Skill 分析智能体）**：识别公文字段，调用公文排版技能生成 Word。
  5. **参数提取节点**：从 Skill 返回结果中提取 file_path、file_name、status、message。
  6. **判断器节点**：根据 status 判断生成成功或失败。
  7. **工具库节点**：调用 MaxKB OSS 通用文件上传，上传生成的 Word 文件。
  8. **指定回复节点**：成功时返回下载链接，失败时返回错误信息。

- **依赖技能**：
  - **公文排版技能 (gongwenformat-pro)**：基于 GB/T 9704-2012 标准的公文排版 Python 脚本，支持标题、主送机关、落款、日期、红头机关、发文字号、份号、密级、紧急程度、签发人、抄送、印发机关、印发日期、附注等完整排版参数。

- **依赖工具**：
  - **MaxKB OSS 通用文件上传**：通过 MaxKB 平台 OSS 接口上传文件，返回临时下载链接（30 分钟有效）。

## 4. 使用说明

1. 将智能体模板导入 MaxKB 平台。
2. 确保已挂载**公文排版技能**和**MaxKB OSS 通用文件上传**工具。
3. （可选）对接知识库以增强公文内容质量。
4. 配置 MaxKB OSS 上传参数（upload_url、upload_headers 等，填入你的 MaxKB 服务地址和 API Token）。
5. 在对话框中输入公文主题或粘贴已有正文，系统自动生成并提供 Word 下载链接。
